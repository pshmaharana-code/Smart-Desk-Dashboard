# 🖥️ J2 MULTI-DASH 
### A Minimalist Smart Desk Display for Spotify & Weather

This project transforms an old Samsung Galaxy J2 into a high-end, three-panel desk dashboard. It features a custom Flask backend, real-time Spotify synchronization (with lyrics!), a live weather API, and a reactive "Pixel Pet."

## ✨ Features
- **Triple-Panel Interface:** Divided layout for Pet, Weather/Lyrics, and Music.
- **Spotify Integration:** Real-time playback status and synced lyrics using LRCLIB.
- **Smart Sleep Mode:** Automatically detects when the host PC is off and enters a dark "System Sleeping" state.
- **Responsive Design:** Optimized for legacy mobile browsers (Fully Kiosk Browser).

## 🛠️ Tech Stack
- **Backend:** Python / Flask
- **Frontend:** HTML5 / CSS3 (Grid/Flexbox) / JavaScript (Async/Fetch)
- **APIs:** Spotify Web API, Open-Meteo, LRCLIB

## 📦 Physical Build
The physical enclosure is designed to hide the phone's bezels, creating an "Infinity Display" look using Industrial Grey housing with Copper accents.
